# Documentation Directory

This directory contains general documentation related to pylink improvements, evaluations, and troubleshooting that are not tied to specific GitHub issues.

## Contents

### Pull Request Documentation

- **[README_PR_fxd0h.md](README_PR_fxd0h.md)**: Documentation for pull request improving RTT auto-detection for nRF54L15 and similar devices. Includes motivation, problem analysis, solution details, and testing results.

### Evaluations and Analysis

- **[RTT2PTY_EVALUATION.md](RTT2PTY_EVALUATION.md)**: Evaluation of replicating `rtt2pty` functionality using pylink. Analyzes capabilities, limitations, and implementation approaches.

### Testing Documentation

- **[test_rtt_connection_README.md](test_rtt_connection_README.md)**: Documentation for RTT connection test script. Includes usage instructions, features, and troubleshooting tips.

### Troubleshooting

- **[TROUBLESHOOTING.md](TROUBLESHOOTING.md)**: General troubleshooting guide for common pylink issues and solutions.
